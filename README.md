# to-schedule-or-not

## Description 
This webpage is designed to help a person organize their daily tasks by the hour. Simply fill out the corresponding to-do box with the time you wish to complete said task and click the save icon. The color of the hour tells you which hour is current, which have passed, and which hours you can still use cram in those other to-do's you forgot about!
